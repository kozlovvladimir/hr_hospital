HR Hospital - Управління лікарнею

Версія: 17.0.1.0.1

Автор: Odoo School

Основні можливості:

  • Управління лікарями, пацієнтами та прийомами.
  • Збереження історії хвороб і діагнозів пацієнтів.
  • Генерація звітів для лікарів.
  • Ієрархічне збереження типів захворювань.
  • Автоматичний контроль записів прийомів.

Візуалізації:

  • Календар прийомів.
  • Графіки та зведені таблиці по діагнозах.
  • Kanban-дошка лікарів.

Як користуватися?

Після встановлення модуля перейдіть до меню "Hospital Management", щоб розпочати роботу.
